Mini Tools Box - 微信小程序工具箱

项目简介：
- 一个微信小程序的工具集合，当前包含图像相关工具：长图拼接、图片加水印、证件照处理页。

主要功能：
- 长图拼接：支持多图选择、拖拽排序、背景色与自定义 HEX，1080 宽高清绘制与预览，导出相册。
- 图片加水印：仅文字水印；大小按原图宽百分比（1%~20%）、透明度、单个/平铺、水平/斜向、平铺密度与最小间距；保留原图分辨率导出。
- 证件照处理：规格 1 寸（295×413）/ 2 寸（413×579），缩放与位置调节，背景颜色选择，背景替换容差可调，导出目标规格 PNG。

目录结构：
- 主包：pages/home/index 首页入口
- 组件：components/tool-grid、components/tool-card
- 分包：subpackages/image-tools/pages
  - index：图像工具箱入口
  - stitch：长图拼接
  - watermark：图片加水印
  - idphoto：证件照处理

环境与配置：
- 使用微信开发者工具导入项目目录
- AppID 可用 touristappid 进行预览，发布请替换为你的 AppID
- 基础库版本参考 project.config.json（3.9.0）
- 权限：scope.writePhotosAlbum 用于保存导出图片/PDF

快速开始：
1) 打开微信开发者工具，导入本项目根目录
2) 运行项目，进入首页 → 图像工具箱
3) 体验对应工具：
   - 图片加水印：选择图片 → 设置文字、大小百分比、透明度、模式与方向、密度与最小间距 → 预览 → 保存图片
   - 长图拼接：选择多图 → 拖拽排序 → 设置背景 → 预览/导出
   - 证件照：选择照片 → 选择规格 → 调整缩放与位置 → 设置背景（可替换白底）→ 导出 PNG

注意事项：
- 背景替换为简单阈值法，容差过高可能影响高亮细节，请按素材适度调整
- 导出保持目标规格或原图分辨率（视工具而定），确保与预览一致
